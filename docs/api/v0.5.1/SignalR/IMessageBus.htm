Task<MessageResult> GetMessages(IEnumerable<string> eventKeys, string cursor, CancellationToken timeoutToken)

Returns a list of messages after the specified cursor or waits for new messages until the timeout is fired if there's no messages.
Parameters
IEnumerable<string> eventKeys
A list of events to wait for messages.
string cursor
A cursor representing how much data the caller has already seen.
CancellationToken timeoutToken
A CancellationToken that represents the timeout of the wait operation when there's no messages.
Returns
Task<MessageResult>
A Task`1 that completes when data is received on the IMessageBus.

string eventKey

object value
The value to send.
Returns
Task
